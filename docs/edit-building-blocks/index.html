    Edit Building Blocks

    Edit building blocks, including adding sections and controls

    Apr 2, 2019

    This page explains how to edit building blocks and the options for using some of the basic and intermediate features of forms4health, such as adding text and number fields, adding data field controls and using reference data to further customise the response options available in your building blocks. What you can find on this page:

    • Building block menu and What would you like to do?
    • Building block structure, settings and general editing
    • Add data field controls
    • Preview a building block

    For more information on advanced features of building blocks, go to the advanced features of forms4health page.

    Building block menu

    The left hand menu in the Building Block Designer contains a range of options for customising building blocks.

    Functions: what would you like to do?

    • Add explanatory text to explain a building block, a section or a data item and help users understand what to do
    • Get users to select one option from a range of options
    • Get users to select-multiple options from a range of options
    • Record a date or time
    • Add images into a data field
    • Use reference data
    • Add a response hint to make a suggestion to users about what data to input
    • Preview a building block

    Add explanatory text

    You might want to add a block of text to explain to a user how to approach completing your form. It’s best to keep this separate from your data fields and you can use the Explanatory Text option from the menu to input and edit your text.

    Use for: adding text that is not part of your data item(s)

    Select one option

    If you want a user to select one option from a range of options, there are a number of ways that you can present the available options to the user. Each one is explained below, or see the overview table.

    Dropdown Menu

    The Dropdown Menu allows you to create a constrained list view of options that users can choose from. This is option can conserve on-screen space within your building block or form and can be useful when the range of options are familiar to the user.

    Use for: small and medium-sized option lists (e.g. up to 20 available options). For example:

    1. Select Dropdown Menu from the left hand Building Block Menu.

    2. Add a relevant label to describe the list or range of options (e.g. Patient’s current GP Practice). Click the List icon on the right hand side of the data item to being editing the Dropdown Menu choices.

    3. The default list contains three labels. Edit the labels to describe the available options. Click the + sign at the top left of the table to add another option at the end of the list or click the arrows on the left hand side to insert a new option above or below an existing option. The arrows can also be used to move an option up or down in the list or remove any unwanted labels. Ensure that each labelled option has a different numeric value in the value column to ensure selected options are coded appropriately. Once you have completed editing, click Apply.

    Radio Buttons

    Radio buttons allow users to quickly choose one option (text or number) from a small range of predefined mutually exclusive options, whilst having the advantages of providing a visual cue to users about the option they have selected and emphasising the other available options.

    Use for: a small list (e.g. no more than 5 options) or a small number range (<10 numbers). For example:

    1. Select Radio Buttons from the left hand Building Block Menu.

    2. Add a relevant label to describe the range of options (e.g. In an average week, how often does the patient forget to take their preventative medication?). Click the List icon on the right hand side of the data item to being editing.

    3. The default list contains three labels. Edit the labels to describe the available options. Click the + sign at the top left of the table to add another option at the end of the list or click the arrows on the left hand side to insert a new option above or below an existing option. The arrows can also be used to move an option up or down in the list or remove any unwanted labels. Ensure that each labelled option has a different numeric value in the value column to ensure selected options are coded appropriately. Once you have completed editing, click Apply.

    Select Buttons

    Select buttons are an alternative way to present a small list (text or numeric) where you want each options to be visible within a clickable button.

    Use for: a small list (e.g. no more than 5 options) or a small number range (<10 numbers). For example:

    1. Follow steps 1-3 to edit radio buttons, as above. Then, click the Control Settings (cog) icon.

    2. Under Basic Settings change the Style dropdown option to ‘Display as Buttons’. Click Apply.

    Slider

    This is a simple way to get a user to select a single numeric value from a small or large number range using a dynamic slider.

    Use for: small (<10 numbers) or large (>10 numbers) number ranges.

    1. Select Slider from the left hand Building Block menu.

    2. Add a relevant label to explain the scale to the user (as well as explanatory text, if required). Click the Control Settings (cog) icon to edit the lower and upper labels on the scale.

    3. Click the Additional Settings tab to edit the:

      • Lower value text with associated Minimum value
      • Upper value text with asscoiated Maximum value
      • Show value as tooltip option - select ‘Yes’ to display the user-selected value on the slider (as in the preview above), or
      • Show value option to display the user-selected value below the slider. After editing, click Apply.
