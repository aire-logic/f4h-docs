Create, add or remove a building block

This section explains the basic building block functions that you will need to build a forms4health form:

  • Create a building block
  • Add a building block to a form
  • Remove a building block
  • Rename a building block

Create a building block

  1. Login to forms4health
  2. Go to the Manage Building Blocks tab and click Create New

  1. The Building Block Designer will open in a new tab. Complete your building block description and click Continue.

    You are now ready to customise your building block. You can find out more about this on the edit building blocks page.

Add a building block to a form

There are two ways to add a building block to a form:

  1. from the Building Block Designer - useful if you want to customise a new building block and then add it into a form.

  2. from the Forms Editor - useful if you have already customised and imported a building block and you want to add it straight into a new or exisintg form. Learn more on the create and manage forms page.

    Note: you must import building blocks to the Forms Editor before you can add them to a new or exisitng form.

Add a building block to a new form from the Building Block Designer

  1. After you create and edit a building block in the Building Block Designer, go back to the Manage Building Blocks tab.
  2. Find the building block(s) you want to add to a form, click the import button and select import. The building blocks will now be available in the Forms Editor under the Manage Forms tab.

  1. To add the imported buliding block into a form straight away, click the add to a form quick-link, which takes you to the Manage Forms tab. You can learn more about building and editing forms on the create and manage forms page.

  1. To add the new building block into a new form, click Create New and complete the New Form information in the dialogue box that opens. Name your new form and select module (forms4health instance) and form type (usually single pass). When complete, click Create.

  1. Complete the Add Page dialogue box by naming the page title and selecing the building block from the drop-down list of imported building blocks. Click Add to add the building block to the new page.

  1. If you want to add another building block, click _add another building block and select from the drop down list of imported building blocks.

  1. From here you can Save, Preview or Discard the form.

To find out more about how to edit forms go to the create and manage forms page.

Remove a building block

  1. To remove a building block, click the delete button next to the building block in the forms editor as shown below.

Rename a building block

  1. To rename a building block, go into the Manage Building Blocks tab, locate the building block you want to rename and on the same row click Import. Then from the drop-down menu click Edit.

  1. In the Building Block Designer, click the building block settings (spanner icon) in the top right-hand corner of the page. This will open the general building block settings where you can change the building block name and title.
